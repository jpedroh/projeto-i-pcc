Instruções para compilação:
A compilação é realizada através do Makefile disponibilizado na raiz do projeto. Para compilar a ferramenta, é suficiente executar `make`. O programa compilado estará disponível para uso no diretório `bin`, e pode ser invocado utilizando `./bin/pmt`
